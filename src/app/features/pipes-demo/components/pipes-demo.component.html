1. Built-in Pipes

Angular provides several built-in pipes for common data transformations.

String Pipes

{{ "message | uppercase" }}
{{ message | uppercase }}
{{ "message | lowercase" }}
{{ message | lowercase }}
{{ "message | titlecase" }}
{{ message | titlecase }}
{{ "longText | slice:0:20" }}
{{ longText | slice : 0 : 20 }}...

Number and Currency Pipes

{{ "price | number" }}
{{ price | number }}
{{ 'price | number:"1.2-2"' }}
{{ price | number : "1.2-2" }}
{{ "price | currency" }}
{{ price | currency }}
{{ 'price | currency:"EUR"' }}
{{ price | currency : "EUR" }}

Date Pipes

{{ "currentDate | date" }}
{{ currentDate | date }}
{{ 'currentDate | date:"short"' }}
{{ currentDate | date : "short" }}
{{ 'currentDate | date:"fullDate"' }}
{{ currentDate | date : "fullDate" }}
{{ 'currentDate | date:"MM/dd/yyyy h:mm a"' }}
{{ currentDate | date : "MM/dd/yyyy h:mm a" }}

JSON and Async Pipes

{{ "products[0] | json" }}
{{ products[0] | json }}

Note: The async pipe is used with Observables and Promises to automatically handle subscriptions and updates.

2. Simple Custom Pipe (TitleCase)

A custom pipe that improves upon the built-in titlecase pipe by ignoring small words (like 'the', 'and', etc.) except at the beginning of the title.

Implementation

Examples

{{ "Built-in: message | titlecase" }}
{{ message | titlecase }}
{{ "Custom: message | appTitleCase" }}
{{ message | appTitleCase }}

3. Transform Pipe with Parameters

A pipe that transforms a price by applying a discount and formatting options.

Implementation

Examples

{{ "price | priceTransform" }}
{{ price | priceTransform }}
{{ "price | priceTransform:0.2" }} (20% discount)
{{ price | priceTransform : 0.2 }}
{{ "price | priceTransform:0.1:false" }} (10% discount, no currency)
{{ price | priceTransform : 0.1 : false }}

4. Filter Pipe

A pipe that filters an array of products based on search criteria.

Implementation

Filter Demo

{{ product.name | appTitleCase }}

{{ product.price | priceTransform }}

{{ product.price | priceTransform : 0.1 }} (10% off)

Category: {{ product.category | titlecase }}

{{ product.inStock ? "In Stock" : "Out of Stock" }}

Released: {{ product.releaseDate | date : "mediumDate" }}

5. Pipe Chaining

Pipes can be chained together to perform multiple transformations in sequence.

Examples

{{ "longText | slice:0:30 | uppercase" }}
{{ longText | slice : 0 : 30 | uppercase }}
{{ "price | currency | lowercase" }}
{{ price | currency | lowercase }}
{{ "products[0].name | appTitleCase | slice:0:10" }}
{{ products[0].name | appTitleCase | slice : 0 : 10 }}...
{{ 'currentDate | date:"MMM dd, yyyy" | uppercase' }}
{{ currentDate | date : "MMM dd, yyyy" | uppercase }}
